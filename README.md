# PhotoAndVideoEncoding
Программа для конвертирования BMP-изображений в YUV420 формат, с последующим наложением полученного изображения на каждый кадр видеоряда YUV420 видео.
Для работы программы в конфигурационном файле необходимо указать подряд названия изображения и видео, а также разрешение этого видео.
В качестве системы сборки выбрана Makefile, а компилирование происходило при помощи компилятора gcc.
